# Household Energy Consumption Analysis Dashboard

## Project Overview

This project presents an interactive **Household Energy Consumption Analysis Dashboard** built in **Microsoft Excel**. The dashboard analyzes household electricity usage by transforming raw energy data into meaningful insights through interactive visualizations and Key Performance Indicators (KPIs).

It helps identify energy-intensive appliances, understand room-wise power consumption, monitor monthly electricity usage trends, and evaluate overall household energy performance.

## Objectives

- Analyze household electricity consumption patterns.
- Identify high power-consuming appliances.
- Compare room-wise energy distribution.
- Monitor monthly electricity usage trends.
- Present key energy metrics using an interactive dashboard.

##  Dashboard Features

- KPI Cards
  - Average Monthly Bill
  - Average Monthly Energy Consumption
  - Highest Wattage Appliance

- Device-wise Power Consumption (Watts)

- Average Device Operating Hours

- Room-wise Power Consumption Distribution

- Monthly Electricity Consumption Trend

- Interactive Slicers for Dynamic Filtering

## Tools & Technologies

## Key Insights

- Identified appliances with the highest power consumption.
- Analyzed average operating hours of household devices.
- Compared electricity usage across different rooms.
- Monitored monthly consumption trends to identify usage patterns.
- Presented energy metrics using interactive visualizations for better decision-making.
